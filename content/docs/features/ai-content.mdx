The template editor includes **AI Assist** — describe what the notification should say and get copy optimized for each channel's character limits.

## BYOP AI keys

Connect your own key in **Integrations**:

- **OpenAI** (`openai_ai`) — uses `gpt-4o-mini`
- **Anthropic** (`anthropic_ai`) — uses `claude-3-haiku`

Nexus does not charge for AI tokens — you pay your provider directly.

## API

```http
POST /v1/templates/ai-generate
Authorization: Bearer <dashboard JWT>
```

```json
{
  "prompt": "Order shipped notification with tracking link",
  "tone": "friendly",
  "locale": "en",
  "channels": ["email", "sms", "push", "whatsapp", "slack"]
}
```

Returns structured content per channel with limits enforced:

| Channel | Limit |
|---------|-------|
| Email subject | 80 chars |
| SMS body | 160 chars |
| Push title / body | 50 / 120 chars |
| WhatsApp body | 1024 chars |

**Tones:** `professional`, `casual`, `urgent`, `friendly`

## Workflow

1. Open **Templates** in the dashboard
2. Click **AI Assist** panel
3. Enter prompt and select channels
4. Preview generated copy per channel
5. Accept into your template draft
