## Rutas SDK (clave secreta)

```http
Authorization: Bearer sk_prd_…
```

| Método | Ruta | Acción |
|--------|------|--------|
| GET | `/v1/schedules/sdk` | Listar schedules |
| POST | `/v1/schedules/sdk` | Crear schedule |
| PATCH | `/v1/schedules/sdk/:id` | Actualizar schedule |
| DELETE | `/v1/schedules/sdk/:id` | Eliminar schedule |

### Cuerpo de creación

```json
{
  "name": "Weekly report",
  "workflowId": "uuid",
  "cronExpression": "0 9 * * 1",
  "timezone": "America/Chicago",
  "recipients": [{ "externalId": "user_1" }],
  "data": { "reportType": "weekly" },
  "active": true
}
```

Schedule puntual — usa `scheduledAt` en lugar de `cronExpression`:

```json
{
  "scheduledAt": "2026-07-15T10:00:00.000Z"
}
```
