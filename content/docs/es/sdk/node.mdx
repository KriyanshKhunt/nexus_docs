## Inicialización

```ts title="nexus.ts"
import { NexusClient } from "@nexus-signal/node";

const nexus = new NexusClient({
  secretKey: process.env.NEXUS_SECRET_KEY!, // sk_prd_…
  baseUrl: "https://api.nexussignal.dev",
  timeout: 5000,
});
```

## Disparar un flujo

```ts title="trigger.ts"
const result = await nexus.workflows.trigger({
  workflowName: "billing.invoice_overdue",
  recipients: [
    {
      externalId: "cust_8824",
      email: "billing@enterprise.com",
      phoneNumber: "+15550198234",
      timezone: "America/New_York",
      locale: "en",
      slackWebhookUrl: "https://hooks.slack.com/services/…",
    },
  ],
  data: {
    invoiceId: "INV-2026-0049",
    amountDue: "$1,250.00",
  },
});

if (!result.ok) {
  console.warn(result.error, result.queued ? "(buffered for retry)" : "");
}
```

## Firmas HMAC

```ts title="auth.ts"
const hmac = nexus.auth.generateHmacSignature("cust_8824");
// Pásalo a NexusProvider de @nexus-signal/react — nunca expongas secretKey en el navegador
```

También exportado de forma independiente (requiere pasar la clave `secretKey` explícitamente):

```ts title="standalone-auth.ts"
import { generateHmacSignature } from "@nexus-signal/node";

const hmac = generateHmacSignature(process.env.NEXUS_SECRET_KEY!, "cust_8824");
```

## Resiliencia

- **Circuit breaker** — se abre tras 5 fallos, se recupera automáticamente tras 60s
- **Retry buffer** — peticiones fallidas en cola en memoria (máx. 500)
- **`flushRetries()`** — reenvía triggers en buffer cuando la plataforma está sana

```ts title="resilience.ts"
const nexus = new NexusClient({
  secretKey: process.env.NEXUS_SECRET_KEY!,
  events: {
    onCircuitOpen: () => console.warn("Nexus circuit open"),
    onCircuitClose: () => nexus.workflows.flushRetries(),
    onBufferOverflow: (p) => console.error("Dropped:", p.workflowName),
  },
});
```

## Forma de respuesta

```ts
type NexusResult<T> = {
  ok: boolean;
  status: number;
  data?: T;
  error?: string;
  queued?: boolean; // true si se almacenó localmente en buffer
};
```

Los triggers exitosos devuelven **HTTP 202** con `{ accepted: true, results: [...] }`.
