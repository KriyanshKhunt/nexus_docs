## Configuración

Cada proveedor acepta credenciales mediante el modal de Integraciones (claves API, URLs de webhook, IDs de remitente). Las claves se cifran en reposo.

Define el costo por mensaje para [analítica de costos](/docs/platform/features/cost-analytics) en **Settings → Provider costs**.
