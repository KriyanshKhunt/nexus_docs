## Core flow

```mermaid
flowchart LR
  A[Your App] -->|POST trigger| B[Express API]
  B -->|202 Accepted| C[BullMQ]
  C --> D[Worker]
  D --> E{User online?}
  E -->|Yes| F[In-App WebSocket]
  E -->|No| G[Email / SMS / Push]
```
