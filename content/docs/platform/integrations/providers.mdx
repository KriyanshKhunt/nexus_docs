## Configuration

Each provider accepts credentials via the Integrations modal (API keys, webhook URLs, sender IDs). Keys are encrypted at rest.

Set per-message cost for [cost analytics](/docs/platform/features/cost-analytics) in **Settings → Provider costs**.
