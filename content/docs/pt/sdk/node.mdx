## Inicialização

```ts title="nexus.ts"
import { NexusClient } from "@nexus-signal/node";

const nexus = new NexusClient({
  secretKey: process.env.NEXUS_SECRET_KEY!, // sk_prd_…
  baseUrl: "https://api.nexussignal.dev",
  timeout: 5000,
});
```

## Acionar um fluxo de trabalho

```ts title="trigger.ts"
const result = await nexus.workflows.trigger({
  workflowName: "billing.invoice_overdue",
  recipients: [
    {
      externalId: "cust_8824",
      email: "billing@enterprise.com",
      phoneNumber: "+15550198234",
      timezone: "America/New_York",
      locale: "pt",
      slackWebhookUrl: "https://hooks.slack.com/services/…",
    },
  ],
  data: {
    invoiceId: "INV-2026-0049",
    amountDue: "R$ 1.250,00",
  },
});

if (!result.ok) {
  console.warn(result.error, result.queued ? "(armazenado em buffer para nova tentativa)" : "");
}
```

## Assinaturas HMAC

```ts title="auth.ts"
const hmac = nexus.auth.generateHmacSignature("cust_8824");
// Passe para o @nexus-signal/react NexusProvider — nunca exponha a secretKey no navegador
```

Também exportado como standalone (requer passar a `secretKey` explicitamente):

```ts title="standalone-auth.ts"
import { generateHmacSignature } from '@nexus-signal/node';

const hmac = generateHmacSignature(process.env.NEXUS_SECRET_KEY!, 'cust_8824');
```

## Resiliência

- **Disjuntor (Circuit breaker)** — abre após 5 falhas, auto-recupera após 60s
- **Buffer de nova tentativa** — solicitações com falha são enfileiradas na memória (máx. 500)
- **`flushRetries()`** — reproduz acionadores armazenados em buffer quando a plataforma estiver saudável

```ts title="resilience.ts"
const nexus = new NexusClient({
  secretKey: process.env.NEXUS_SECRET_KEY!,
  events: {
    onCircuitOpen: () => console.warn("Disjuntor do Nexus aberto"),
    onCircuitClose: () => nexus.workflows.flushRetries(),
    onBufferOverflow: (p) => console.error("Descartado:", p.workflowName),
  },
});
```

## Formato de resposta

```ts
type NexusResult<T> = {
  ok: boolean;
  status: number;
  data?: T;
  error?: string;
  queued?: boolean; // true se armazenado localmente em buffer
};
```

Gatilhos bem-sucedidos retornam **HTTP 202** com `{ accepted: true, results: [...] }`.
