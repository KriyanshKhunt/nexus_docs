title: Analytics API
description: Dashboard analytics — overview, traffic, latency, pipeline health, and cost.
---

Requires dashboard JWT + workspace context.

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/v1/analytics/overview` | Summary cards — triggers, success rate, ROI |
| GET | `/v1/analytics/overview/traffic` | Channel traffic time series |
| GET | `/v1/analytics/overview/latency` | Ingestion → delivery latency |
| GET | `/v1/analytics/overview/pipeline` | Pipeline health flow |
| GET | `/v1/analytics/overview/heatmap` | Error heatmap by provider |
| GET | `/v1/analytics/overview/cost` | Provider spend breakdown |

### Query parameters

```
?environment=PRODUCTION&timeframe=24h
```

`environment`: `DEVELOPMENT` | `STAGING` | `PRODUCTION`

### Cost response (excerpt)

```json
{
  "totalCostUsd": 423.18,
  "budgetUsd": 500,
  "budgetUsedPercent": 85,
  "budgetExceeded": false,
  "byProvider": [
    { "providerId": "twilio", "label": "Twilio", "count": 8200, "costUsd": 410 }
  ],
  "byWorkflow": [
    { "workflowName": "auth.otp", "costUsd": 180.5 }
  ]
}
```

Configure provider unit costs via `PUT /v1/settings/provider-costs`.
