## Authentication modes

| Audience | Auth | Endpoints |
|----------|------|-----------|
| Server apps | `Authorization: Bearer sk_*` | `/workflows/trigger`, `/schedules/sdk` |
| Browser SDK | `x-nexus-public-key: pk_*` | `/sdk/*` |
| Dashboard | JWT session cookie / Bearer | `/workflows/dashboard`, `/analytics`, etc. |

## Response conventions

- Triggers return **202 Accepted** (async processing)
- Errors return `{ error: string, code?: string, details?: object }`
- Rate limits apply per plan tier
