## 初期化

```ts title="nexus.ts"
import { NexusClient } from "@nexus-signal/node";

const nexus = new NexusClient({
  secretKey: process.env.NEXUS_SECRET_KEY!, // sk_prd_…
  baseUrl: "https://api.nexussignal.dev",
  timeout: 5000,
});
```

## ワークフローのトリガー

```ts title="trigger.ts"
const result = await nexus.workflows.trigger({
  workflowName: "billing.invoice_overdue",
  recipients: [
    {
      externalId: "cust_8824",
      email: "billing@enterprise.com",
      phoneNumber: "+15550198234",
      timezone: "Asia/Tokyo",
      locale: "ja",
      slackWebhookUrl: "https://hooks.slack.com/services/…",
    },
  ],
  data: {
    invoiceId: "INV-2026-0049",
    amountDue: "¥125,000",
  },
});

if (!result.ok) {
  console.warn(result.error, result.queued ? "（リトライのためバッファ済み）" : "");
}
```

## HMAC 署名

```ts title="auth.ts"
const hmac = nexus.auth.generateHmacSignature("cust_8824");
// @nexus-signal/react の NexusProvider に渡します — ブラウザに secretKey を公開しないでください
```

スタンドアロンとしてもエクスポートされています（`secretKey` を明示的に渡す必要があります）:

```ts title="standalone-auth.ts"
import { generateHmacSignature } from '@nexus-signal/node';

const hmac = generateHmacSignature(process.env.NEXUS_SECRET_KEY!, 'cust_8824');
```

## レジリエンス

- **サーキットブレーカー** — 5回の失敗後に開放、60秒後に自動回復
- **リトライバッファ** — 失敗したリクエストをメモリにキュー（最大500件）
- **`flushRetries()`** — プラットフォームが正常なときにバッファされたトリガーを再送

```ts title="resilience.ts"
const nexus = new NexusClient({
  secretKey: process.env.NEXUS_SECRET_KEY!,
  events: {
    onCircuitOpen: () => console.warn("Nexus サーキット開放"),
    onCircuitClose: () => nexus.workflows.flushRetries(),
    onBufferOverflow: (p) => console.error("ドロップ:", p.workflowName),
  },
});
```

## レスポンス形状

```ts
type NexusResult<T> = {
  ok: boolean;
  status: number;
  data?: T;
  error?: string;
  queued?: boolean; // ローカルにバッファされた場合は true
};
```

成功したトリガーは `{ accepted: true, results: [...] }` で **HTTP 202** を返します。
