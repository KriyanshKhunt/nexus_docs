## Nexus vs 一般的なプラットフォーム

| 機能 | Nexus | 一般的なインフラ |
|------|-------|--------------|
| プレゼンス抑制 | あり | まれ |
| AI スマート送信時刻 | あり | まれ |
| プロバイダーコスト分析 | あり | なし |
| BYOP 手数料ゼロ | あり | よく組み込まれている |
| AI コンテンツ生成 | あり | なし |

## 機能の接続方法

```mermaid
flowchart LR
  T[トリガー] --> P{プレゼンス?}
  P -->|オンライン| I[アプリ内のみ]
  P -->|オフライン| S[スマートタイミング]
  S --> W[配信ウィンドウ]
  W --> C[チャンネルディスパッチ]
  C --> F[フェイルオーバー監視]
```
