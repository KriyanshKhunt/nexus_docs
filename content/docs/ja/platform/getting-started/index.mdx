## 前提条件

- Nexus Signal アカウント（[無料登録](https://app.nexussignal.dev)）
- サーバー SDK 用の Node.js 18 以上
- 少なくとも1つのプロバイダーアカウント（SendGrid、Resend、Twilio など）

## 配信の仕組み

```mermaid
sequenceDiagram
  participant App as あなたのアプリ
  participant API as Nexus API
  participant W as ワーカー

  App->>API: POST /workflows/trigger
  API-->>App: 202 Accepted
  API->>W: ジョブをエンキュー
  W->>W: ワークフローステップ<br/>+ プロバイダー
  W-->>App: WebSocket 経由でアプリ内通知
```

すべての配信はデバッグと分析のために完全なライフサイクル状態でログに記録されます。
